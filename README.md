App-ClockUtils
======================================

#INSTALLATION

To install this module type the following:

    perl Makefile.PL
    make
    make test
    make install

or using cpanm:

> git clone https://github.com/xtetsuji/p5-App-ClockUtils.git

    cpanm p5-App-ClockUtils/
